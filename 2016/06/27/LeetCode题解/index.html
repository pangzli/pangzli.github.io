Reverse String

Write a function that takes a string as input and returns the string reversed.

Example:
Given s = “hello”, return “olleh”.

class Solution {
public:
    string reverseString(string s) {
        if(s == "")return "";
        char temp;
        int len = s.length();
        for(int i=0;i<len/2;i++)
        {
            temp = s[i];
            s[i]=s[s.length()-i-1];
            s[s.length()-i-1] = temp;
        }
        return s;     
    }
};

Nim Game

You are playing the following Nim Game with your friend: There is a heap of stones on the table, each time one of you take turns to remove 1 to 3 stones. The one who removes the last stone will be the winner. You will take the first turn to remove the stones.

可以拿1-3个石子，拿到最后一个石子的人获胜，假设你是第一个拿石子的人。

Both of you are very clever and have optimal strategies for the game. Write a function to determine whether you can win the game given the number of stones in the heap.

For example, if there are 4 stones in the heap, then you will never win the game: no matter 1, 2, or 3 stones you remove, the last stone will always be removed by your friend.

分析：

  • 0 块石子 - false
  • 1 块石子 - true
  • 2 块石子 - true
  • 3 块石子 - true
  • 4 块石子 - false
  • 5 块石子 - true
  • 6 块石子 - true
  • 7 块石子 - true
  • 8 块石子 - false

发现，当是4块时，这时要拿石子的人不管拿1-3中的几块，都会失败

而总数是5块时，可以拿1块让总数变成4块，总数是6块时，可以拿2块让总数变成4块…..

以此类推，发现当4的倍数时失败，其余情况成功。

class Solution {
public:
    bool canWinNim(int n) {
        if(n%4 == 0)
        return false;
        else 
        return true;
    }
};

Single Number

Given an array of integers, every element appears twice except for one. Find that single one.

给定一个整形数组，只有一个数字出现了一次，其余的都出现了两次，找到这个只出现了一次的数组。

这个题第一次看到的时候，我是这样做的：

class Solution {
public:
    int singleNumber(vector<int>& nums) {

        int temp;

        for (int i=0;i<nums.size();i++)
        {
            temp = nums[i];
            bool flag = false;
            for(int j=0;j<nums.size();j++)
            {
                if(j==i)continue;
                if(temp == nums[j])
                flag = true;
            }

            if(flag == false)
                break;

        }
            return temp;
    }
};

时间复杂度为O(n^2)，结果是：Time Limit Exceeded

后来发现有两个公式：

  • a ⊕ a = 0
  • a ⊕ 0 = a

可以写出一段特别简单的代码：

class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int temp=0;
        for(int i=0;i<nums.size();i++)
        {
            temp = temp xor nums[i];
        }
        return temp;  
    }
};

时间复杂度为O(n)。

Add Digits

Given a non-negative integer num, repeatedly add all its digits until the result has only one digit.

For example:

Given num = 38, the process is like: 3 + 8 = 11, 1 + 1 = 2. Since 2 has only one digit, return it.

我用了递归的方法，当结果是两位数时继续相加，当为个位数是开始返回。

class Solution {
public:
    int addDigits(int num) {
        if(num>=10)
            return addDigits(num/10+num%10);
        else
            return num;
    }
};

后来，在网上找更优的解法，发现下面这个算法。

// Let's observe the pattern
//    1    1
//    2    2
//    ... ...
//    8    8    
//    9    9    
//    10    1
//    11    2
//    12    3    
//    ... ...
//    17    8
//    18    9
//    19    1
//    20    2
//    ...  ...
// It looks most of number just simply %9 is the answer, 
// but there are some edge cases.
//    9%9=0 but we need 9. 
//    18%9=0 but we need 9
// so we can find the solution is:
//    1) num <=9, return num
//    2) num > 9, reutrn num%9 if num%9>0
//                return 9 if num%9 ==0
int addDigits03(int num) {
    return num >9 ? ((num %9)==0 ? 9:num%9) : num;
}

//But actually, we can use (num-1)%9 + 1 to make all cases right.
int addDigits04(int num){
    return (num - 1) % 9 + 1;
}

//This solution is similar with pervious solution.
int addDigits05(int num){
    return num - 9 * ((num - 1)/9);
}

上面这三个算法，都是同一种思路，发现当给定的数不是9的倍数时，结果是该数%9的结果，否则结果是9。

Maximum Depth of Binary Tree

Given a binary tree, find its maximum depth.

The maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node.

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
 * };
 */
class Solution {
public:
    int maxDepth(TreeNode* root) {

        if(root==NULL)
        return 0;

        int left = maxDepth(root->left);
        int right = maxDepth(root->right);

        if(left>right)
            return left+1;
        else
            return right+1;
    }
};

利用递归的方法，每层的深度是左子树与右子树中高的那个加一。

Reverse Linked List

Reverse a singly linked list.

A linked list can be reversed either iteratively or recursively. Could you implement both?

迭代方法（iteratively）：

/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    ListNode* reverseList(ListNode* head){

        if(head==NULL||head->next==NULL) return head;

        ListNode *newN = head; // 新排序好的第一个
        ListNode *oldN = newN->next; // 待排序的第一个
        head -> next = NULL;  // 将头结点的next清空
        ListNode *tempN = NULL; // 定义一个临时变量存储oldN的下一个

        while(oldN) // 不断将结点的next反向
        {
            tempN = oldN->next;
            oldN->next = newN;
            newN = oldN;
            oldN = tempN;
        }

        return newN;
    }
};

递归方法（recursively）：

/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    ListNode* reverseList(ListNode* head){
        if (head==NULL || head->next==NULL) return head;

        ListNode *h = reverseList(head->next);
        head->next->next = head;
        head->next = NULL;

        return h;
    }
};

递归方法简单易懂。

Counting Bits

Given a non negative integer number num. For every numbers i in the range 0 ≤ i ≤ num calculate the number of 1’s in their binary representation and return them as an array.

Example:
For num = 5 you should return [0,1,1,2,1,2].

计算从0到num一共num+1个数，每个数的二进制形式中1的个数，组成一个数组。

class Solution {
public:
    vector<int> countBits(int num) {
        vector<int> temp(num+1, 0);

        for (int i=0;i<num+1;i++)
        {
            temp[i]=numOf1(i);
        }

        return temp;

    }
    int numOf1(int n) // 剑指offer书里介绍的一种非常巧妙地计算一个二进制数中1的个数的算法
    {
        int count = 0;
        while(n)
        {
            ++ count;
            n = n & (n-1);
        }

        return count;
    }
};

Invert Binary Tree

Invert a binary tree.

     4
   /   \
  2     7
 / \   / \
1   3 6   9

to

     4
   /   \
  7     2
 / \   / \
9   6 3   1

递归算法：

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
 * };
 */
class Solution {
public:
    TreeNode* invertTree(TreeNode* root) {
        if(root==NULL)return root;

        TreeNode *temp = invertTree(root->left); // 将左边的保存
        root->left = invertTree(root->right); // 左右互换
        root->right = temp;

        return root;   
    }
};
